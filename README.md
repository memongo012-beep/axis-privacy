# Política de Privacidad — Axis

**Última actualización:** 21 de abril de 2026

## Información que recolectamos

Axis NO recolecta, almacena ni transmite ningún dato personal ni de uso a servidores externos. Toda la información que creas en la aplicación (clases, ejercicios, duraciones personalizadas, configuraciones) se almacena exclusivamente de manera local en tu dispositivo usando el sistema de almacenamiento del sistema operativo.

## Información que NO recolectamos

- No recolectamos nombres, emails, números de teléfono ni ninguna información de identificación personal.
- No rastreamos tu ubicación geográfica.
- No utilizamos servicios de analítica ni herramientas de seguimiento de usuarios.
- No compartimos datos con terceros porque no los recolectamos.
- No utilizamos cookies ni identificadores publicitarios.

## Permisos del dispositivo

Axis puede solicitar los siguientes permisos del sistema operativo:

- **Vibración / Haptics:** para proporcionar feedback táctil durante el uso del cronómetro en el Modo Instructor.
- **Mantener pantalla encendida:** para que la pantalla del dispositivo no se apague automáticamente durante una clase en vivo.

Estos permisos no recolectan información personal y se utilizan únicamente para mejorar la experiencia dentro de la aplicación.

## Retención de datos

Los datos generados por el usuario (clases guardadas, ejercicios personalizados) se mantienen en el dispositivo mientras la aplicación esté instalada. Al desinstalar Axis, todos los datos se eliminan automáticamente junto con la aplicación.

## Menores de edad

Axis está diseñada como herramienta profesional para instructoras de Pilates y Barre. No está dirigida a menores de 13 años.

## Cambios a esta política

Esta política puede actualizarse periódicamente. Cualquier cambio se publicará en esta misma página con la fecha de actualización modificada.
